문의 채널 문의 템플릿 자주 묻는 질문

1. 문의 채널

  • GitHub 이슈: https://github.com/makemeblue41-sudo/Codex-lotto/issues
  • 정책/광고 문의: 이슈 제목 앞에 [정책문의]를 붙여 접수해 주세요.
  • 개인정보가 포함된 문의는 민감정보를 제외하고 요약 형태로 남겨 주세요.

2. 빠른 처리를 위한 문의 템플릿

  • 문의 유형: 오류 제보 / 데이터 문의 / 정책 문의 / 기타
  • 발생 시점: YYYY-MM-DD HH:mm (KST)
  • 재현 방법: 클릭 순서 또는 입력값
  • 기대 결과와 실제 결과: 가능한 한 구체적으로 작성
  • 환경 정보: 브라우저 종류, 모바일/PC 여부

3. 자주 묻는 질문

  • Q. 번호 생성 결과가 당첨을 보장하나요?
    A. 아닙니다. 본 사이트는 확률 정보와 번호 조합 도구를 제공합니다.
  • Q. 통계 값이 실제 회차와 다른 것 같아요.
    A. 이슈로 접수해 주시면 원본 데이터와 계산 로직을 확인해 수정합니다.
  • Q. 정책 문서를 인용해도 되나요?
    A. 출처 표시 조건에서 가능합니다. 상업적 재배포는 별도 문의가 필요합니다.
